docker学习笔记

Posted on 2022-04-23

一、概念

docker是一个软件容器。是简易Linux系统

二、三要素

镜像：是一个固定的模板。可以类比成java中的类

容器：镜像的实例。可以看做java中的对象

仓库：集中存放镜像的地方。仓库和仓库注册服务器是有区别的。仓库注册服务器上存放着多个仓库，每个仓库又包含了多个镜像，每个镜像有不同的标签（tag),仓库又分为公开仓库和私有仓库。

五、docker客户端命令

5.1、帮助命令

--查看docker版本
docker verion
--docker的描述
docker info
--查看docker的命令有哪些
docker --helpe

5.2、镜像命令

--查看当前主机有哪些镜像实例
docker images
参数说明：
docker images -a 列出本地所有镜像层（包含中间层）
docker images -q 只显示镜像id
docker images --digests 显示镜像的摘要信息

--查看某个镜像的名字
docker search 镜像的名字
--下载镜像
docker pull 镜像的名字 : 镜像的id（不带参数就是最新版）
--删除某个镜像,加上-f就是强制删除
docker rmi -f 镜像的名字 : 镜像的id（不带参数就是最新版）
